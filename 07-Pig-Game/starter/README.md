Project 3: Pig Game

It is a code a long practice project building a game.
How the game works is that player one roll the dice and the number rolled gets added to current score for that round. This is the same for player two.
Both player can select hold score at any tome to add the current score to the total score. Then the players also switch turns.
If players one rolls 1 the player looses all their points and their current score goes back to zero. Then it is the other player turn. Total scores stays the same.
The first player who reaches a score of 100 wins.
