Project 3: Pig Game

It is a code a long practice project building a game.
How the game works is that player one roll the dice and the number rolled gets added to current score for that round. This is the same for player two.
Both player can select hold score which adds the value to the total score. The we switch players.
If one of the players roll a 1 they looses all their current score and their current score goes back to zero. Total scores stays the same.
The first player who reaches a score of 100 wins.
