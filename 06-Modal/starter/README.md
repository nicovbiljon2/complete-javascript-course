Project 2 Modal Window:

In this project we going to build a UI component (modal window).
We will use JavaScript to add functionality to the buttons so that when it is click the overlay modal window will pop up. You can then also close the window by clicking the x button or outside the modal window.

Added functionality to the buttons when clicked the modal page is visible by using the classList.remove() method to remove the hidden class and then closing the modal page by adding the hidden class back using the classList.add() method. The same for the overlay class.
Add a keypress event that closes the modal window when the escape button is pressed.
